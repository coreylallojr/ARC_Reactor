Capabilities

Everything JARVIS Was Built For

Voice Narration

Every Claude Code action — file reads, edits, bash commands — is narrated in JARVIS's dry British voice via a custom Piper TTS model trained on Paul Bettany's performance.

3D Arc Reactor

A real-time Three.js sphere of 152 nodes pulses with audio frequency data. Bloom post-processing, dual shells, orbital rings, and signal pulses visualize your session's activity live.

Voice Conversation

Speak directly to JARVIS using Web Speech API. He classifies intent — answering questions instantly via Ollama, or delegating code tasks to Claude Code and reporting back.

Text Chat Console

Pull up the slide-in console panel to chat with JARVIS in text. Full conversation history with streaming token display and session-aware context.

Proactive Intelligence

JARVIS checks in when you've been idle for 45 seconds, celebrates every 25-tool-call milestone, notices repeated errors, and volunteers commentary on long sessions — unprompted.

Session Memory

Conversations persist across sessions at ~/.claude/jarvis-memory/. JARVIS summarises completed sessions via Ollama and recalls previous context the next time you open Claude Code.

Installation Steps

  1. Download arc-reactor-setup.exe from the button above
  2. Run the installer — it auto-installs silently to %LOCALAPPDATA%\Programs
  3. ARC Reactor launches automatically and appears in your system tray
  4. On first run, the setup wizard detects Python and installs the Piper TTS model
  5. Click Launch JARVIS — the neural UI opens in your browser at localhost:7474
  6. Open Claude Code in any project — JARVIS starts narrating immediately

System Requirements

Windows 10/11 x64
Claude Code CLI
Python 3.10+ for TTS
