Overview
--------

ARC Reactor is an AI agent companion layer that rides inside `Claude Code <https://claude.ai/code>`_.
It intercepts every tool call your AI makes, generates a contextual JARVIS voice line via a local LLM,
synthesizes it with a neural voice model trained on Paul Bettany's MCU performance, and plays it through
a 3D browser visualization that reacts to the actual audio waveform in real time.

**You use Claude Code normally. JARVIS rides along.**
