◈   INSTALL HOOKS

JARVIS hooks into Claude Code to narrate every action.
This writes two entries to ~/.claude/settings.json — fully reversible.

It also adds a jarvis command to your shell profile so you can type
jarvis start from any terminal to launch everything.
